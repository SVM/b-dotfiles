
# Version 1.1
------
添加了 中文 man 手册支持；
使用方法：
```shell
cman cd
```
github源：[https://github.com/lidaobing/manpages-zh][1]

### 目录结构
----
~/mydotfiles
	config
	install.sh
	b-man-zh
		install-man-zh
    b-shell
        install-shell.sh
        bashrc
        bash_profile
    b-vim
        install-vim.sh
        vimrc
        vimrc.bundles
    b-tmux
        install-tmux.sh
        tmux.conf
        tmux.conf.local
    b-powerline
        install-powerline.sh

[1]: https://github.com/lidaobing/manpages-zh
